1.ラッキーナンバーチェッカー3について

1-1.吉凶の種類

番号評価説明
1大吉とても有利な状況でよい結果が出ることが予想される。
2吉有利な状況で良い結果が出ることが予想されるうえ、さらなる発展の可能性もある。
3平2つの道の前に立っている状況を表し、
自分の努力や選択によって、
有利な状況にも不利な状況にもなり得る転換点を意味します。
4凶不利な状況が予想されます。何かを変えれば不利な状況を改善することができます。
5大凶とても不利な状況が予想されますが、自分の短所を直すことができれば
人生を幸せなものにすることができます。

●ここで気を付けなければいけないのは、
元々、凶兆(きょうちょう)を暗示する数字があるわけではないということです。
各人の悪い心や短所のせいで凶兆(きょうちょう)を暗示する数字になってしまうだけで、 凶の数字が出たからといって、必ずしも良くないことが起こってしまうわけではありません。

2.開発の動機

2-1.動機

このアプリは爻象（こうぞう）と言って、３桁までの数字(0～999)を入力してその数字の吉凶を判定するアプリです。

我々が、日常生活をしていると、世の中は数字に溢れています。何気なく数字を使ったとき、この数字は、吉なのか凶なのか、無意識に気になりませんか？

まあ。凶であっても吉であっても、それほど大したことではないのですが。。。。
ということを心の奥底に密（ひそか）に気に留めていました。

私が生活をしていて、そういう数字の吉凶を求めるという物を本屋で発見しました。
それをネタにアプリを作ってみました。
